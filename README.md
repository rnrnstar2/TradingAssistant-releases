# TradingAssistant-releases

Public release assets for [TradingAssistant](https://github.com/rnrnstar2/TradingAssistant).

This repository hosts updater manifests (`latest.json`) and signed update bundles for all profiles:

| Profile | Tag | Manifest |
|---------|-----|----------|
| full | `latest-full` | `releases/download/latest-full/latest.json` |
| latency | `latest-latency` | `releases/download/latest-latency/latest.json` |
| mt-only | `latest-mt` | `releases/download/latest-mt/latest.json` |
